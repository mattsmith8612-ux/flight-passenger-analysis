# Flight Passenger Analysis

Exploratory analysis of historical airline passenger data using Python, pandas and matplotlib.

## Project Overview

This project investigates trends in airline passenger numbers over time using a historical flights dataset. The analysis focuses on identifying long-term trends and seasonal patterns in passenger demand.

## Skills Demonstrated

- Python
- pandas
- Datetime handling with `pd.to_datetime()`
- Data aggregation with `groupby()`
- Data visualisation with `matplotlib`
- Basic time-series analysis

## Questions Investigated

- How have passenger numbers changed over time?
- What is the average number of passengers per year?
- Which months have the highest average passenger numbers?
- Are there any clear seasonal patterns in air travel demand?

## Files

- `flight_passenger_analysis.py` – Main analysis script.
- `flights.csv` – Dataset used for the analysis.
- `flight_passengers_over_time.png` – Passenger numbers over time.
- `average_passengers_per_year.png` – Average passenger numbers by year.
- `average_passengers_per_month.png` – Average passenger numbers by month.

## Libraries Used

- matplotlib
